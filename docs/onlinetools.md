# sslip.io
sslip.io это служба DNS (Domain Name System), которая при запросе имени хоста со встроенным IP-адресом возвращает этот IP-адрес. Она была вдохновлена xip.io, которая была создана Сэмом Стивенсоном.

<details>
<summary>Для чего нужен сервис sslip.io, как им пользоваться</summary>

Сервис sslip.io предоставляет простой способ использовать DNS для тестирования локальных веб-приложений. Он автоматически преобразует IP-адрес в доменное имя, что позволяет обходить проблемы с DNS, когда требуется тестирование на локальных или временных серверах.
</details>

<https://sslip.io>

# What2Log
What2Log - это проект, который начался, когда мы увидели заметный пробел в обозначениях компьютерной безопасности в отношении журналов. Не было централизованного ресурса о том, что и как понимать в журналах основных операционных систем, которые используются каждый день. What2Log был создан именно для того, чтобы заполнить этот пробел.

<https://what2log.com>

<https://attack.mitre.org>
